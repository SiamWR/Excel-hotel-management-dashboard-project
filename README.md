# Excel-hotel-management-dashboard-project
Interactive Hotel Management Dashboard built using Microsoft Excel to analyze revenue, booking trends, cancellation rates, lead times, and distribution channels.
## Objectives
- Analyze hotel revenue performance
- Monitor cancellation trends
- Evaluate booking channels
- Assess lead time impact
- Track yearly booking performance
- ## Key Metrics

- Total Revenue
- Revenue Loss
- Cancellation Rate
- Total Bookings
- Average Daily Rate (ADR)
